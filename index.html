Eco-friendly books & art cards as community keys (NFTs). This demo shows a walletless, gas-free UX for judges: claim a proof, download it, and see how it anchors to decentralised storage (Lighthouse/IPFS) — roadmap to Hedera for low-fee minting.

Showcase: claim a **demo** eco-NFT (no blockchain fee) → downloads NFT metadata + local proof file you can show to judges.
Important: This demo does not mint on-chain. It simulates a low-friction, walletless claim flow and produces a download-ready metadata file that is IPFS / Lighthouse ready.

Why this demo suits Soulbloomers

  • Zero barrier: users can claim a proof without a wallet and later link it to a real wallet.

Roadmap: Magic (passwordless wallets) → Yellow/Thirdweb paymaster (gasless UX) → Hedera HTS for low-fee anchoring.
